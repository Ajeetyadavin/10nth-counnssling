# Career Counselor Quiz App

This project consists of a high-engagement Frontend and a robust Admin Backend.

## 📁 Structure
- `/frontend`: React + Vite application (The Quiz)
- `/backend`: Node.js + Express API (The Admin Data Store)

## 🚀 Getting Started

### 1. Run the Backend
```bash
cd backend
npm install
# Start in Dev mode
npm run dev
```
*Port: 5000*

### 2. Run the Frontend
```bash
cd frontend
npm install
npm run dev
```
*Port: 5173*

## 🛠 Features
- **Partial Sync:** Student data is saved as soon as they fill the lead form.
- **Admin HQ:** Accessible via the landing page footer.
- **PDF Generation:** Admin can download professional reports for completed tests.
- **CSV Export:** One-click export for all student leads.
